# 地图导航功能配置指南

## 概述
本应用使用**腾讯地图 WebService API** 进行路线规划、地点搜索等功能。腾讯地图是微信小程序原生地图组件的底层提供商，集成度最高，可靠性最强。

## 当前状态
- ✅ 导航页面已完全重写，支持腾讯地图API路线规划
- ✅ 地点搜索功能可用（关键字搜索、逆地理编码）
- ✅ 微信导航（`wx.openLocation`）作为备选方案
- ✅ 开发者工具和真机都支持

## 快速开始

### 1. 申请腾讯地图API Key

访问 [腾讯地图开放平台](https://lbs.qq.com/) 申请免费API Key：

1. 注册账号并登录
2. 进入 **开发者中心** → **应用管理**
3. 点击 **创建应用**，填写应用名称
4. 在应用详情页，点击 **创建密钥**，选择 **WebService API** 类型
5. 保存生成的 API Key（格式如 `IXHBZ-BHHKU-DHWVZ-BMAHF-BSXU3-YNBA3`）

### 2. 配置白名单

在腾讯地图后台，为应用配置白名单：

1. 进入应用详情页
2. 找到 **Key 管理** → **白名单设置**
3. 添加以下白名单：
   - `servicewechat.com`（微信小程序服务）
   - 或选择 **不校验白名单**（不推荐，但便于开发测试）

### 3. 更新API Key

编辑文件 `pages/map/navigatePage/navigatePage.js`，找到以下行：

```javascript
var TENCENT_MAP_KEY = 'IXHBZ-BHHKU-DHWVZ-BMAHF-BSXU3-YNBA3';
```

替换为你的API Key：

```javascript
var TENCENT_MAP_KEY = '你的API_Key';
```

## 功能说明

### 路线规划
- **支持的出行方式**：驾车（推荐）、步行、骑行、公交
- **默认策略**：LEAST_TIME（耗时最少）
- **返回信息**：距离、时间、详细路线

### 地点搜索
- **搜索范围**：盐池县（可修改 `region` 参数）
- **返回结果**：位置名称、坐标、地址

### 逆地理编码
- **功能**：坐标 → 地址
- **用途**：显示当前位置的真实地址

## 常见问题

### Q: 路线规划失败，显示"未找到路线"？
**A**: 可能原因：
1. API Key 无效或白名单配置错误
2. 起点或终点坐标不在中国范围内
3. API 配额已用完（免费版每天 25000 次请求）

**解决方案**：
- 检查 API Key 是否正确
- 确保坐标在有效范围内（中国境内）
- 联系腾讯地图技术支持

### Q: 为什么不用百度地图？
**A**: 
- 腾讯地图是微信小程序 `<map>` 组件的原生提供商
- 集成度高，兼容性更好
- 不需要额外的数据转换
- 百度地图API在小程序中容易被禁用（CORS/域名问题）

### Q: 开发者工具中无法使用真实导航功能？
**A**: 
- 这是正常的，开发者工具的模拟环境不支持 `wx.openLocation`
- 使用真机调试可以完整测试导航功能
- 在模拟环境中会显示友好的提示信息

### Q: 如何修改搜索区域？
**A**: 编辑 `searchAndPlan()` 和 `selectStartLocation()` 方法中的 `region` 参数：

```javascript
qqmapsdk.search({
  keyword: endText,
  region: '宁夏回族自治区',  // 修改为其他区域
  success: function (res) { ... }
});
```

## 性能优化建议

1. **缓存常用地点**：避免重复搜索
2. **限流**：添加搜索防抖（debounce）
3. **异步处理**：使用 Promise 管理异步流程
4. **错误恢复**：路线规划失败时自动降级到微信导航
