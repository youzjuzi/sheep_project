## 📞 常见问题

### Q1: 优惠券不显示怎么办？
**A:** 
1. 检查是否有优惠券创建（Web 后台创建）
2. 检查优惠券状态是否为"激活"
3. 检查优惠券有效期是否符合条件
4. 查看浏览器控制台是否有 API 错误

### Q2: 地图导航无法规划路线
**A:**
1. 确保已申请腾讯地图 API Key
2. 确保已配置白名单（servicewechat.com）
3. 检查 API Key 是否正确填入代码
4. 查看是否超过每日配额（免费版 25000 次/天）
5. 尝试使用"直接导航"功能（微信原生导航）

### Q3: 小程序无法获取当前位置
**A:**
1. 检查 app.json 是否配置了位置权限
2. 检查手机是否已授予小程序定位权限
3. 确保 GPS 已开启
4. 在开发者工具中模拟位置进行测试

### Q4: Web 后台进不去
**A:**
1. 确保已登录管理员账号
2. 检查 URL 是否正确：`/admin/coupon_list/`
3. 检查是否有相应的访问权限
4. 查看服务器日志是否有错误
